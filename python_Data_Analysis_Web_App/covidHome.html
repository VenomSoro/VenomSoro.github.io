This is a simple web app that allows for easy interaction with the graphs I generated while messing around with
this dataset from the CDC. These visuals were made using Python and various libraries for data analysis
such as Pandas, PySpark, Matplotlib, and Seaborn. With some experience using SQL to query excel sheets,
databases, and other forms of tabular data, I was able to use PySpark to query for information and create
these tables using a combination of Matplotlib and Seaborn.

The script along with a GUI Application to use the script can be found here.
